Welcome!

This is a demo application for a fictitious hiking club for memebers to view and edit their profiles & memberships. There is also a page listing the board of directors and the group's committee members. A simple ticketing system for managing data corrections & updates (and other improvements) is under development.

Built with Django 3.2.5, Python 3, Javascript ES6, JQuery 3.2.1, Bootstrap 4/5, HTML5, CSS3

Hosted on AWS Lightsail with Bitnami Django 3.2.5-8
